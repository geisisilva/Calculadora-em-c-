# Calculadora em C#

Projeto desenvolvido em C# como prática de lógica de programação. A aplicação permite realizar operações matemáticas básicas (adição, subtração, multiplicação e divisão) por meio da entrada de expressões matemáticas no console.

## Funcionalidades

* Soma (+)
* Subtração (-)
* Multiplicação (*)
* Divisão (/)
* Execução de múltiplos cálculos em sequência
* Tratamento de divisão por zero
* Encerramento do programa mediante comando do usuário

## Tecnologias Utilizadas

* C#
* .NET
* Console Application

## Objetivo

Aplicar conceitos fundamentais de programação, como estruturas de repetição, condicionais, manipulação de strings, entrada e saída de dados e tratamento de erros.
